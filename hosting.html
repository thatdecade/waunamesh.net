Host a node

Have a good location? You can help Waunakee connect.

A useful host site can be a home, farm, barn, church, business, school, or community building with power, height, or a clear view across town.

Talk with the community Learn how the network works

Better coverage

Your location may bridge a gap and help messages travel across Waunakee.

Emergency resilience

The mesh can keep local messages moving when internet or cell service is unavailable.

Low effort

Once installed, a fixed node can run quietly with little or no action from the host.

What hosting means

A small radio in a useful place.

Hosting usually means allowing a small weatherproof radio box and antenna to be placed where it can hear other nodes. Outdoor, elevated, and clear-view locations are most valuable.

Common host locations

  • Homes with a good view or upper floor window
  • Barns, farms, and outbuildings
  • Churches, schools, and community buildings
  • Local businesses with roof or window access
  • High spots near the edge of current coverage

Typical equipment

  • Small MeshCore compatible LoRa radio
  • Weatherproof enclosure for outdoor installs
  • Small antenna mounted with permission
  • USB power or solar power where appropriate
  • Owner contact details for maintenance

Host questions and answers

Does it need my internet?

No. A MeshCore radio can pass local radio messages without using your Wi-Fi or internet connection.

Does it use much electricity?

No. These are small, low-power radios. Many fixed nodes can run from a USB power adapter, and some locations can use solar power.

Will it interfere with Wi-Fi, TV, or cell service?

It is not expected to. The radios use low-power LoRa in the 902 to 928 MHz ISM band in the United States, which is different from normal Wi-Fi and cellular frequencies.

Is a license required?

No license is required for typical MeshCore LoRa mesh devices in this project.

Who maintains it?

The community member or team placing the node should handle setup, updates, and removal. Hosts should not need to manage the software.

Can I ask for it to be removed?

Yes. A host should be able to ask for the node to be moved or removed at any time.

What data passes through it?

Mesh nodes relay short radio messages from the local mesh network. A host node is not a connection to your home network.

What makes a location valuable?

Height, a clear view, reliable power, and a position that fills a coverage gap all help. Schumacher Farm Park is a great example of why elevation and line of sight matter.
